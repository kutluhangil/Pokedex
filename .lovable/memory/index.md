# Project Memory

## Core
- Stack: React, Vite, Tailwind CSS, Framer Motion. Data: PokeAPI with local caching.
- UX/Performance: 60fps spring animations, memoization, retina-optimized official artwork, lazy loaded images, no layout shifts.
- UI Interactions: Strict z-index management. All decorative elements (silhouettes, particles) must use `pointer-events-none`.
- Keyboard: `/` Dex, `?` shortcuts panel, `S` shiny, `F` favorite, `←/→` nav between Pokémon, `Esc` close. Skip in inputs.
